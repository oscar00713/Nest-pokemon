 # Ejecutar en Desarrollo

 1.-clonar el proyecto 
 2.-ejecutar

 ```
 yarn install
 ```
 3.Tener nes CLI instalado

 ```
 npm i g @nestjs/cli
 ```

 4. levantar la DB

```
docker-compose up -d
```

## Stack Usado 
*mongoDB
*NestJS
